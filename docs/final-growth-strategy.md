## Thumbnail + Title Formula

### Thumbnail: 5 Non-Negotiable Rules
1. **Character face: 60% of frame minimum.** Single large face. Exaggerated expression — surprise or joy.
2. **High-contrast background.** Saffron orange, deep royal blue, or lime green. No dark palettes.
3. **1–3 words maximum.** Tamil script + English on the same thumbnail. No sentence-length text.
4. **Single story-hook prop.** One object that suggests the story: glowing lamp, giant mango, oversized Pongal pot.
5. **No clutter.** No gradients, shadows, or text crossing the character's face.

**What fails:** Multiple characters, dark/muted colors, text-heavy designs, generic stock-art look identical to hundreds of competitors.

### Title Formulas (by content type)

**Formula A — Festival/Seasonal:**
```
[Number] [festival] traditions [character] loves | [Tamil script] | Tamil cartoon for kids
```
Example: `5 Puthandu traditions Kavin loves | பொங்கல் கதை | Tamil cartoon for kids`

**Formula B — Character Story (diaspora-optimized):**
```
[Character]'s [surprising/emotional thing]! | Tamil animated story for kids
```
Example: `Kavin finds a magical mango tree! | Tamil animated story for kids`

**Formula C — Educational Micro-content:**
```
Learn [X] Tamil words with [Character] | Tamil language for kids | [Tamil script]
```
Example: `Learn 5 Tamil words with Kavin | Tamil language for kids | குழந்தைகள் தமிழ்`

**Formula D — India/LK optimized (serve subcontinent Shorts audience):**
```
[Tamil script title] | [Topic] | Tamil [Content Type] for Kids
```
Example: `கரடி மாமா கதை | Karadi Mama | Tamil Stories for Kids`

**SEO rules for every title:**
- "Tamil" within the first 3 words
- At least one Tamil script phrase (captures Tamil-keyboard searches from India + Singapore)
- End with category anchor: "Tamil cartoon for kids" OR "Tamil animated story"
- Total title under 60 characters (mobile truncation threshold)
- One primary keyword per video — do not stack

**A/B Test Protocol:**
- Upload 2–3 thumbnail variants at publish time using YouTube's Test & Compare
- 72h minimum before selecting winner
- Select by highest CTR × highest average view duration — not CTR alone
- Log which visual elements (expression type, color, prop) correlate with wins — build a channel-specific style guide after 10 tests
